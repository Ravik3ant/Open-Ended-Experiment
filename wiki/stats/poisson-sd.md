# Poisson SD (`sqrt(mu)`)

## Poisson property

For Poisson random variable `X ~ Poisson(lambda)`:

- `E[X] = lambda`
- `Var(X) = lambda`

Therefore:

- `SD(X) = sqrt(lambda)`

Using sample estimate `mu` for `lambda`:

- `Poisson SD estimate = sqrt(mu)`

## Importance

Primary model-consistency check: observed SD near `sqrt(mu)` indicates Poisson-like counting.
